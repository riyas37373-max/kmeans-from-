# K-Means Clustering from Scratch

**Overview:**  
A step-by-step project to implement K-Means clustering using only NumPy, visualize clusters, and compare performance with scikit-learn.

**Steps:**
- Generate synthetic data
- Run custom K-Means algorithm
- Visualize clusters and centroids
- Compute inertia
- Discussion and comparison

## Files
- main.py — core implementation
- report.txt — project summary, findings
- clusters.png — result visualization
